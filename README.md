# WorldRecipes

## About the project
WorldRecipes Built up with ReactJs with backend ExpressJS and deployment on Vercel.

## Usage 

### Clone
```shell
git clone https://github.com/karomahlailatul/WorldRecipes.git
```
### Installing 
```shell
npm i;
```

### Testing
```shell
npm run start;
```
